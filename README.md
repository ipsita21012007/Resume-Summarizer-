# Resume-Summarizer-
📄 AI Resume Summarizer
A smart web application that automatically analyzes resumes and extracts key insights using AI and natural language processing.

🚀 Quick Start
1. Install & Run
bash
# Create virtual environment
python -m venv resume_env
resume_env\Scripts\activate  # Windows
# source resume_env/bin/activate  # Mac/Linux

# Install dependencies
pip install streamlit python-docx PyPDF2 pdfplumber

# Run the app
streamlit run app.py
2. Open in Browser
Navigate to http://localhost:8501

✨ Features
📄 Multi-Format Support: PDF, DOCX, TXT files

🤖 AI Analysis: Automatic section detection and summarization

🔍 Smart Extraction: Contact info, skills, education, experience

💡 Insights: Personalized improvement suggestions

📊 Analytics: Word count, section analysis, metrics

🎯 How to Use
Upload your resume file (PDF/DOCX/TXT)

Or paste resume text directly

Click Analyze to process

View results with extracted information and insights

📋 Supported Content
Contact Information (email, phone)

Professional Summary

Work Experience

Education Details

Skills & Technologies

Certifications & Projects

🔧 Tech Stack
Frontend: Streamlit

Processing: python-docx, PyPDF2, pdfplumber

AI: OpenAI GPT (optional)

Analysis: Custom NLP algorithms

Perfect for job seekers, recruiters, and career coaches!
